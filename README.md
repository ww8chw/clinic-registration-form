# 安安診所掛號單（HTML 可編輯版）

## 檔案結構

```
掛號單/
├── index.html              ← 入口頁（選擇內科 / 婦產科）
├── form-internal.html      ← 內科掛號單
├── form-obgyn.html         ← 婦產科掛號單
├── styles.css              ← 版面樣式 + 列印 @page 設定
├── editor.js               ← 共用：點選編輯 + 自動儲存 + 模式切換
└── assets/
    ├── logo-horizontal.png ← 橫式 logo（入口頁用）
    ├── logo-symbol.png     ← 只有圖形的 logo（掛號單背景浮水印）
    └── logo.png            ← 原始轉檔（備用）
```

## 怎麼開啟

雙擊 `index.html`，或從終端機執行：
```
open ~/Claude/掛號單/index.html
```

## 兩種修改內容的方式

### 方式 A：在瀏覽器上直接點選修改（日常使用）

打開任一張掛號單後，把滑鼠移到任何文字（選項、標題、連「號碼」「日期」都行），背景會變黃、滑鼠變文字游標 → **直接點下去就能改字**。

- 修改**立即自動存**到這台瀏覽器的 localStorage
- A4 雙拼的上下兩份會**即時同步**（改一邊，另一邊跟著變）
- 按 Enter 會結束編輯（避免多插空行）
- 想全部還原 → 按右上角「**還原預設內容**」按鈕

> ⚠ 此方式只存在**當前瀏覽器+當前裝置**。換電腦或清瀏覽器資料就沒了。

### 方式 B：直接修改 HTML 原始檔（永久改、所有裝置都生效）

用文字編輯器（VS Code、記事本…）打開 `form-internal.html` 或 `form-obgyn.html`，找到：

```html
<template id="form-body">
  ...
  <li><span class="chk"></span><span class="editable" data-k="c1">高血壓</span></li>
  ...
```

- 改字：把 `高血壓` 換成想要的文字
- 新增一個選項：複製整行 `<li>...</li>` 貼到下一行，修改文字，**`data-k` 要給一個沒用過的 key**（例如 `c9`、`c10`）
- 刪除選項：整行 `<li>...</li>` 砍掉

`data-k` 是儲存 key，只要不改名，舊的修改紀錄都留著；改掉就會當作新欄位處理。

## 下載 PDF

1. 右上角選好 **A5 單張** 或 **A4 雙拼**
2. 點「**下載 PDF**」按鈕，約 1–2 秒會自動下載 PDF 檔到「下載」資料夾
3. 檔名格式：`內科掛號單_a5_2026-04-18.pdf`

> 實體列印時：
> - 把下載好的 PDF 用預覽程式打開
> - 列印時紙張尺寸請匹配（A5 模式選 A5、A4 模式選 A4）
> - 邊界設「無」或「最小」
> - 若印表機匣只有 A4，請在網頁上切到 **A4 雙拼** 模式再下載

## 內科掛號單設計重點

依診所客群調整（**健檢慢性病人 + 減重** 為主，感冒/外傷次要）：

1. **基本量測** 改為六格：體溫、血壓、身高、體重、BMI、腰圍
2. **今天就診主因**（大方向，病人先勾這個）：慢性病追蹤 / 減重諮詢 / 健康檢查 / 感冒 / 外傷 / 其他
3. **慢性病史**（針對慢性病追蹤 & 健檢）：高血壓、糖尿病、高血脂、心血管、腎臟、甲狀腺、痛風、其他
4. **感冒／急性症狀**（只有當次為感冒/外傷才勾）：發燒、咳嗽、鼻水、喉嚨痛、嘔吐、腹瀉、腹痛、外傷
5. **特殊需求**：維持原本 5 項

## 版面調整

全部在 `styles.css` 開頭 `:root` 區塊：

```css
:root {
  --stub-width: 46mm;       /* 左側存根聯寬度 */
  --page-pad-x: 4mm;        /* 左右內距（離紙邊） */
  --page-pad-y: 3mm;        /* 上下內距 */
  --fs-body: 13pt;          /* 主要文字大小 */
  --fs-label: 13pt;         /* 選項文字 */
  --fs-title: 14pt;         /* 區塊標題 */
  --fs-stub: 16pt;          /* 存根聯文字 */
  --chk-size: 4mm;          /* checkbox 方框 */
}
```

背景浮水印大小在：
```css
.form-page::before {
  background-size: 105mm auto;   /* 改這個數字 */
  opacity: 0.08;                 /* 透明度 */
}
```

- 婦產科 A4 雙拼：`form-obgyn.html?mode=a4-duo`
